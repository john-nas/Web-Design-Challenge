Weather Analysis

Summary

The Weather Analysis project involved analysing data from 500+ cities across the world with varying proximity to the equator. To complete the analysis a variety of platforms have been utilised:

OpenWeatherMap API:
API was utilised to pull live weather data from Open Weather

Matplotlib:
was used to create all the plots to present all of the below variables:

Temperature (C), Cloudiness (%), Humidity (%) and Wind Speed (km/h).

VS Code:
Editing/Compiling Python,HTML,Bootstrap CSS,JavaScript files.

HTML,Bootstrap,JavaScript:
The final presentation has been displayed in this website
